## üõ†Ô∏è Setup Instructions

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. **Clone or Download** the portfolio files to your local machine
2. **Open** the `index.html` file in your web browser
3. **Customize** the content with your personal information

### Local Development Server (Optional)

For the best development experience, you can use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (if you have it installed)
npx serve .

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

## üé® Customization Guide

### Personal Information
1. **Name**: Replace `[Your Name]` in the About section
2. **Contact Details**: Update email, phone, and location in the Contact section
3. **Social Links**: Add your actual social media profiles
4. **Photo**: Replace the placeholder with your professional photo

### Content Updates
1. **Projects**: Update the project cards with your actual work
2. **Research**: Modify the research section with your publications
3. **Skills**: Adjust the skill tags to match your expertise
4. **About Text**: Personalize the about section content

### Styling
- **Colors**: Modify the CSS custom properties for different color schemes
- **Fonts**: Change the Google Fonts import for different typography
- **Layout**: Adjust grid layouts and spacing in the CSS file
